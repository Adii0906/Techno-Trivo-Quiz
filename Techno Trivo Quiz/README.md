# Techno Trivo Quiz

## Overview

Welcome to the Techno Trivo Quiz, a Streamlit-based app for testing knowledge in the Basics of Computer Science. This quiz comprises Multiple Choice Questions (MCQs) designed to assess the understanding of fundamental concepts.

## Getting Started

### Prerequisites

- Make sure you have Python installed on your machine.

### Installation
pip install streamlit

1. Clone the repository:

   ```bash
   git clone https://github.com/Adii0906/Techno-Trivo-Quiz..git
Navigate to the project directory:

bash
cd techno-trivo-quiz
Install dependencies:

bash
pip install -r requirements.txt
Usage
Run the Streamlit app:
streamlit App.py
Open your browser and navigate to http://localhost:8501 to start the quiz.

Features
Multiple Choice Questions (MCQs) on Basics of Computer Science.
Interactive and user-friendly interface.
